# About Brain Code Camp

**Brain Code Camp** เป็นส่วนหนึ่งของโครงการ **BrainCode101** ซึ่งเป็นกิจกรรมที่เน้นการปฏิบัติจริง (Hands-On Workshop) ผ่านแพลต์ฟอร์มออนไลน์ โดยมุ่งเน้นการพัฒนาทักษะและความเข้าใจให้กับผู้เรียนในด้านการเขียนโปรแกรมเพื่อทำความเข้าใจความสัมพันธ์ระหว่างพฤติกรรมและสัญญาณสมองผ่านการสร้างแบบจำลองทางคณิตศาสตร์และปัญญาประดิษฐ์

โครงการนี้ได้รับการสนับสนุนจากหน่วยบริหารและจัดการทุนด้านการพัฒนากำลังคน และทุนด้านการพัฒนาสถาบันอุดมศึกษา การวิจัยและการสร้างนวัตกรรม (บพค.)
