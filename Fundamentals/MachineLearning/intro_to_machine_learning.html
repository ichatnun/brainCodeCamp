Machine learning#

Machine learning (การเรียนรู้ของเครื่อง) คือสาขาหนึ่งของปัญญาประดิษฐ์ (artificial intelligence) ที่เน้นการพัฒนาและออกแบบวิธีการให้เครื่องคอมพิวเตอร์สามารถเรียนรู้จากข้อมูลโดยไม่จำเป็นต้องระบุโครงสร้างหรือกฎกำหนดอย่างชัดเจน เครื่องคอมพิวเตอร์ที่ใช้การเรียนรู้เองจะสามารถวิเคราะห์และสกัดความรู้หรือแบบแผนที่อยู่ภายในชุดข้อมูลขนาดใหญ่ ทำนายผลลัพธ์ หรือทำงานอื่นได้ โดยจะต้องอาศัยอัลกอริทึม (Algorithms) ต่างๆ ในการเรียนรู้

AI-Machine Learning-Deep Learning: Machine Learning เปรียบเสมือน subset ของ AI เเล้ว Deep Learning เองก็เป็นเสมือนวิธีการหนึ่งของ Machine Learning เช่นกัน ที่มา

กระบวนการหลักใน machine learning ประกอบด้วยขั้นตอนต่อไปนี้:

  1. การเก็บรวบรวมข้อมูล (Data collection): ขั้นตอนแรกคือการรวบรวมข้อมูลที่เกี่ยวข้อง ข้อมูลนี้อาจเป็นข้อความ รูปภาพ ค่าตัวเลข หรือข้อมูลที่มีโครงสร้าง

  2. การตรวจสอบและจัดเตรียมข้อมูล (Data preprocessing): เมื่อมีข้อมูลแล้วจะต้องทำการตรวจสอบและจัดเตรียมข้อมูลให้พร้อมใช้งาน ในขั้นตอนนี้ได้แก่ การจัดการข้อมูลที่หายไป การแปลงข้อมูลให้เป็นรูปแบบที่เหมาะสม

  3. การแยกแยะลักษณะ (Feature extraction): ในขั้นตอนนี้จะแปลงข้อมูลให้อยู่ในลักษณะที่เอาไปใช้งานได้ใน machine learning เช่น การแปลงข้อมูล text หรือรูปภาพ ให้อยู่ในรูปแบบตัวเลข

  4. การเลือกแบบจำลอง (Model Selection): โดยพิจารณาความเหมาะสมของปัญหาและข้อมูลที่มีอยู่ จะเลือกแบบจำลองที่เหมาะสมต่อการใช้งาน เช่น แบบจำลองการทำนายค่า (regression) แบบจำลองการจำแนกประเภท (classification) หรือแบบจำลองการจัดกลุ่มข้อมูล (clustering)

  5. การฝึกแบบจำลอง (Model Training): ในกระบวนการนี้จะเป็นการให้ข้อมูลที่เตรียมแก่แบบจำลองที่เราเลือก เพื่อฝึกฝนและเรียนรู้

  6. การประเมินแบบจำลอง (Model Evaluation): เมื่อแบบจำลองถูกฝึกแล้ว จะต้องทำการประเมินผลประสิทธิภาพและความสามารถในการใช้งานของแบบจำลอง โดยใช้ชุดข้อมูลทดสอบที่แยกไว้เพื่อประเมินค่าเชิงสถิติเช่นความแม่นยำ (accuracy) ค่าความแม่นยำเพิ่ม (precision) ค่าความแม่นยำลด (recall) หรือค่าความคลาดเคลื่อนเฉลี่ย (mean squared error)

ทั้งหมดนี้เป็นแนวทางขั้นตอนพื้นฐานใน machine learning และเราสามารถใช้ไลบรารีหรือเครื่องมือต่างๆในภาษา Python เช่น scikit-learn, TensorFlow, หรือ PyTorch เพื่อสร้างและฝึกแบบจำลอง machine learning ได้

การจำแนก Machine learning ตามการเรียนรู้ของข้อมูล#

หลักการเรียนรู้ข้อมูลมีอยู่หลายประเภท เช่น

  • Supervised Learning (การเรียนรู้แบบมีผู้สอน): ใน Supervised Learning, โมเดลถูกสอนด้วยชุดข้อมูลที่มีคำตอบ (labeled data) ซึ่งมีการกำหนดผลลัพธ์ให้กับข้อมูลทุกตัวอย่าง โมเดลจะเรียนรู้และสร้างรูปแบบในการจำแนกหรือทำนายผลลัพธ์ของข้อมูลใหม่ตามความสัมพันธ์ระหว่างตัวแปรต้นและตัวแปรตาม โมเดลที่เป็นที่นิยมใช้สำหรับ Supervised Learning ได้แก่ Linear Regression, Logistic Regression, Support Vector Machines (SVM), Decision Trees, Random Forests, Neural Networks และอื่น ๆ

  • Unsupervised Learning (การเรียนรู้แบบไม่มีผู้สอน): ใน Unsupervised Learning, ลักษณะของการเรียนรู้นี้เป็นแบบที่ไม่มีการกำหนด Output ของ Data แต่จะเป็นในรูปแบบของการให้ Data กับโมเดล จากนั้นโมเดลจะเรียนรู้โดยการสำรวจโครงสร้างและลักษณะเด่นของข้อมูล โมเดลจะพยายามหาความสัมพันธ์และโครงสร้างในข้อมูลโดยไม่มีการตอบสนองหรือคำตอบใด ๆ ที่ให้มา อัลกอริทึมที่ใช้ในส่วนของ Unsupervised Learning ได้แก่ Clustering (การจัดกลุ่ม), Dimensionality Reduction (การลดมิติของข้อมูล เพื่อลดความซับซ้อนก่อนนำไปใช้ต่อ หรือเพื่อแสดงผลในรูปกราฟที่คนอ่านได้)

  • Reinforcement Learning (การเรียนรู้แบบเสริมกำลัง): ใน Reinforcement Learning, โมเดลสามารถเรียนรู้และพัฒนาตนเองได้จากการลองผิดลองถูก และสังเกตสิ่งแวดล้อมรอบตัว ภายใต้แนวคิดที่ว่าจะเลือกกระทำสิ่งที่ทำให้ได้ผลลัพธ์มากที่สุด ตัวอย่างของ Reinforcement Learning ที่ถูกนำมาประยุกต์ใช้ เช่น AlphaGo ที่สามารถเล่นเกมโกะให้ชนะผู้เล่นระดับโลก

ภาพแสดงการจำแนก Machine learning ตามการเรียนรู้ของข้อมูล: Supervised learning, Unsupervised learning และ Reinforcement learningที่มา

โดยที่ใน tutorial ส่วนนี้จะเน้นไปที่ Supervised learning ในการทำนายผลลัพธ์ของการเรียนรู้ชนิดนี้สามารถแบ่งได้เป็น

Classification(การจำแนกประเภท): เป็นการทำนายผลลัพธ์ที่มีลักษณะเป็นค่าแบบกลุ่มหรือประเภท โดยตัวอย่างที่นิยมใช้คือการจำแนกอีเมล์เป็นอีเมล์ที่เป็นสแปมหรือไม่เป็นสแปม หรือการจำแนกภาพถ่ายเป็นภาพถ่ายของคนหรือสัตว์

Regression (การทำนายค่า): เป็นการทำนายผลลัพธ์ที่มีลักษณะเป็นค่าต่อเนื่องหรือตัวเลข โดยตัวอย่างที่นิยมใช้คือการทำนายราคาของบ้าน, การทำนายยอดขายสินค้าในอนาคต การวัดผลสามารถใช้ได้ทั้ง ค่า R-squared, Mean Absolute Error (MAE), Mean Squared Error (MSE),ค่า Mean Absolute Percentage Error (MAPE) หรือ RMSE

ในส่วนของ supervised learning จะโฟกัสไปที่ปัญหาทาง classification เพื่อจำแนกประเภทของข้อมูล ซึ่งในส่วนของ regression เพื่อทำนายค่าตัวเลขจากโมเดลนั้นจะมีขั้นตอนต่างๆ ที่คล้ายๆกับ classification models ยกเว้นในส่วนของการ import library ซึ่งสามารถศึกษาเพิ่มเติมได้จาก https://scikit-learn.org/stable/supervised_learning.html

Machine learning - classification ที่มา

#Model evaluation การประเมินโมเดลเป็นหนึ่งในขั้นตอนที่สำคัญในกระบวนการพัฒนาโมเดล เพื่อเป็นการวัดประสิทธิภาพ (Performance) ของโมเดลว่าดีเพียงพอต่อการนำไปใช้งาน สำหรับการประเมินผลของงานทางด้านจำแนกประเภท (classification) สามารถพิจารณาได้จาก accuracy, confusion matrix, precision, recall, F1 score, ROC curve, AUC เป็นต้น มีรายละเอียดดังต่อไปนี้

  • Confusion Matrix ตารางที่แสดงผลลัพธ์จาก machine learning สำหรับปัญหาทางด้าน classification โดยสามารถแสดงเป็นตารางซึ่งประกอบด้วย True Positive(TP),True Negative (TN), False Positvie(FP), และ False Negative (FN)

รูปแสดง Confusion Matrix ที่มา

โดยที่

  • TP: True Positive จำนวนกรณีที่เป็นจริงและทำนายว่าเป็นจริง

  • TN: True Negative จำนวนกรณีที่เป็นเท็จและทำนายว่าเป็นเท็จ

  • FP: False Positive จำนวนกรณีที่เป็นเท็จแต่ทำนายว่าเป็นจริง

  • FN: False Negative จำนวนกรณีที่เป็นจริงแต่ทำนายว่าเป็นเท็จ

  • ค่าความถูกต้อง (Accuracy): เป็นค่าที่แสดงถึงความแม่นยำของโมเดลโดยรวม ใช้บอกว่าโมเดลสามารถทำนายได้ถูกต้องและตรงกับสิ่งที่เกิดขึ้นจริงเท่าไรเมื่อเทียบกับจำนวนการทำนายทั้งหมด โดยค่า accuracy จะมีค่าอยู่ระหว่าง 0-1 ยิ่งเข้าใกล้ 1 แปลว่าโมเดลเราทำนายผลได้ดีมาก \ \(\text{Accuracy} = \frac{\text{TP} + \text{TN}} {\text{TP} + \text{TN} + \text{FP} + \text{FN}}\)

  • ค่าความแม่นยำ (Precision): แสดงถึงความแม่นยำของการทำนายในคลาสที่สนใจหรือคาดหวัง \ \(Precision = \frac{TP}{TP+FP}\)

  • ค่าความอ่อนไหว (Sensitivity/Recall): แสดงถึงความสามารถของโมเดลในการทำนายได้อย่างถูกต้อง เทียบกับผลการทำนายที่ทั้งถูกและผิดของโมเดล \ \(Recall = \frac{TP}{TP+FN}\)

  • ส่วนค่าความจำเพาะ (Specificity) คือ สัดส่วนความถูกต้องของผลการทำนายที่เราไม่ได้สนใจ (True Negative) โดยมีค่าเท่ากับ \(Specificity = \frac{TN}{TN+FP}\)

  • F1 Score: ค่าเฉลี่ยฮาร์โมนิกระหว่าง precision และ recall \ \(F1\text{-}Score = 2 \times \frac{Precision \times Recall}{Precision + Recall}\)

  • ROC Curve (Receiver Operating Characteristic Curve): กราฟที่แสดงความสามารถในการแยกแยะของโมเดลระหว่าง true positive rate (sensitivity) และ false positive rate (1-specificity) เป็นการพล็อตระหว่างค่าสัดส่วนที่ทำนายได้อย่างถูกต้อง กับ ค่าสัดส่วนที่ทำนายผิด (false positive)

  • AUC (Area Under the ROC Curve): ค่าของพื้นที่ใต้เส้นโค้ง ROC ซึ่งสามารถใช้เป็นตัวบ่งชี้ของประสิทธิภาพของโมเดลในการแยกแยะ AUC มีค่าอยู่ระหว่าง 0-1 ยิ่งเข้าใกล้ 1 แปลว่าโมเดลในภาพรวมสามารถทำนาย y ได้ดีมาก TPR: True Positive Rate (sensitivity) \ FPR: False Positive Rate (1-specificity) \ \(TPR = \frac{TP}{TP+FN}\), \ \(FPR = \frac{FP}{FP+TN}\) \ สามารถศึกษาหาข้อมูลเพิ่มเติมเกี่ยวกับได้จาก metrics and scoring
