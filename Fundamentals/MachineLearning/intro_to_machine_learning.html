Machine learning#

Machine learning (การเรียนรู้ของเครื่อง) คือสาขาหนึ่งของปัญญาประดิษฐ์ (artificial intelligence) ที่เน้นการพัฒนาและออกแบบวิธีการให้เครื่องคอมพิวเตอร์สามารถเรียนรู้และปรับปรุงผลลัพธ์ได้โดยอัตโนมัติจากข้อมูล โดยไม่ต้องระบุโครงสร้างหรือกฎกำหนดอย่างชัดเจน เครื่องคอมพิวเตอร์ที่ใช้การเรียนรู้เองจะสามารถวิเคราะห์และสกัดความรู้หรือแบบแผนที่อยู่ภายในชุดข้อมูลขนาดใหญ่ ทำนายผลลัพธ์ หรือทำงานอื่น ๆ ได้ โดยการเรียนรู้นี้จะเป็นไปในรูปแบบการปรับค่าพารามิเตอร์หรือโครงสร้างของแบบจำลอง (model) เพื่อให้สามารถปรับปรุงผลลัพธ์ให้ดีขึ้นตามข้อมูลที่มีอยู่

AI-Machine Learning-Deep Learning ที่มา

กระบวนการหลักใน machine learning ประกอบด้วยขั้นตอนต่อไปนี้:

  1. การเก็บรวบรวมข้อมูล (Data collection): ขั้นตอนแรกคือการรวบรวมข้อมูลที่เกี่ยวข้องและแทนสมบัติของปัญหาที่ต้องการแก้ไข ข้อมูลนี้อาจเป็นข้อความ รูปภาพ ค่าตัวเลข หรือข้อมูลที่มีโครงสร้าง

  2. การตรวจสอบและจัดเตรียมข้อมูล (Data preprocessing): เมื่อมีข้อมูลแล้วจะต้องทำการตรวจสอบและจัดเตรียมข้อมูลให้พร้อมใช้งาน การตรวจสอบและจัดเตรียมข้อมูลรวมถึงการลบข้อมูลเสียงหรือข้อมูลที่ไม่เกี่ยวข้อง การจัดการข้อมูลที่หายไป การปรับข้อมูลให้เป็นรูปแบบที่เหมาะสม หรือการแปลงข้อมูลให้อยู่ในรูปแบบที่เหมาะสม

  3. การแยกแยะลักษณะ (Feature extraction): ในขั้นตอนนี้จะทำการแยกแยะและสกัดคุณ สมบัติหรือลักษณะที่เกี่ยวข้องออกจากข้อมูล คุณสมบัติเหล่านี้จะเป็นข้อมูลที่นำเข้าสู่อัลกอริทึมของ machine learning เพื่อใช้ในการทำนายหรือตัดสินใจ

  4. การเลือกแบบจำลอง (Model Selection): โดยพิจารณาความเหมาะสมของปัญหาและข้อมูลที่มีอยู่ จะเลือกแบบจำลองที่เหมาะสมต่อการใช้งาน เช่น แบบจำลองการทำนายค่าต่อเนื่อง (regression) แบบจำลองการจัดกลุ่ม (classification) หรือแบบจำลองการจัดกลุ่มข้อมูล (clustering)

  5. การฝึกแบบจำลอง (Model Training): แบบจำลองที่เลือกได้จะถูกฝึกด้วยข้อมูลที่ผ่านการแยกแยะและจัดเตรียมแล้ว ในขั้นตอนนี้แบบจำลองจะเรียนรู้และปรับปรุงผลลัพธ์ของมันตามข้อมูลที่ใช้ฝึก

  6. การประเมินแบบจำลอง (Model Evaluation): เมื่อแบบจำลองถูกฝึกแล้ว จะต้องทำการประเมินผลประสิทธิภาพและความสามารถในการใช้งานของแบบจำลอง โดยใช้ชุดข้อมูลทดสอบที่แยกไว้เพื่อประเมินค่าเชิงสถิติเช่นความแม่นยำ (accuracy) ค่าความแม่นยำเพิ่ม (precision) ค่าความแม่นยำลด (recall) หรือค่าความคลาดเคลื่อนเฉลี่ย (mean squared error)

ทั้งหมดนี้เป็นแนวทางขั้นตอนพื้นฐานใน machine learning และเราสามารถใช้ไลบรารีหรือเครื่องมือต่างๆในภาษา Python เช่น scikit-learn, TensorFlow, หรือ PyTorch เพื่อสร้างและฝึกแบบจำลอง machine learning ได้

Machine learning สามารถแบ่งเป็นประเภทต่างๆ ได้ดังนี้#

  • Supervised Learning (การเรียนรู้แบบมีผู้สอน): ใน Supervised Learning, โมเดลถูกสอนด้วยชุดข้อมูลที่มีคำตอบ (labeled data) ซึ่งมีการกำหนดผลลัพธ์ที่ถูกต้องให้กับข้อมูลทุกตัวอย่าง โมเดลจะเรียนรู้และสร้างรูปแบบในการจำแนกหรือทำนายผลลัพธ์ของข้อมูลใหม่ตามความสัมพันธ์ระหว่างตัวแปรต้นและตัวแปรตาม ตัวอย่างของ Supervised Learning ได้แก่ Linear Regression, Logistic Regression, Support Vector Machines (SVM), Decision Trees, Random Forests, Neural Networks และอื่น ๆ.

  • Unsupervised Learning (การเรียนรู้แบบไม่มีผู้สอน): ใน Unsupervised Learning, โมเดลจะเรียนรู้โดยการสำรวจโครงสร้างและลักษณะเด่นของข้อมูล โมเดลจะพยายามหาความสัมพันธ์และโครงสร้างในข้อมูลโดยไม่มีการตอบสนองหรือคำตอบใด ๆ ที่ให้มา ตัวอย่างของ Unsupervised Learning ได้แก่ Clustering (การจัดกลุ่ม), Dimensionality Reduction (การลดมิติข้อมูล), Association Rule Learning (การค้นหาความสัมพันธ์ในข้อมูล) และอื่น ๆ

  • Reinforcement Learning (การเรียนรู้แบบเสริมสร้าง): ใน Reinforcement Learning, โมเดลเรียนรู้โดยการทดลองและสำรวจสิ่งแวดล้อม โมเดลจะเรียนรู้การตอบสนองต่อสิ่งแวดล้อมที่มีการตอบรับเป็นรางวัลหรือล้าหลัง การเรียนรู้ใน Reinforcement Learning ได้รับการกำหนดค่าด้วยการทดลองและรับรู้จากผลลัพธ์ที่ได้ ตัวอย่างของ Reinforcement Learning ได้แก่ Q-Learning, Deep Q-Networks (DQN), Policy Gradient, Actor-Critic และอื่น ๆ. classification/ regression evaluation metrics

ประเภทของ Machine learning ที่มา

โดยที่ใน tutorial ส่วนนี้จะเน้นไปที่ Supervised learning ในการทำนายผลลัพธ์ในการเรียนรู้ของเครื่องจักรนี้มีความแตกต่างกันในเรื่องของประเภทของผลลัพธ์ที่ต้องการทำนาย

Classification(การจำแนก): เป็นการทำนายผลลัพธ์ที่มีลักษณะเป็นค่าแบบกลุ่มหรือประเภท โดยตัวอย่างที่นิยมใช้คือการจำแนกอีเมล์เป็นอีเมล์ที่เป็นสแปมหรือไม่เป็นสแปม หรือการจำแนกภาพถ่ายเป็นภาพถ่ายของคนหรือสัตว์

Regression (การทำนาย): เป็นการทำนายผลลัพธ์ที่มีลักษณะเป็นค่าต่อเนื่องหรือตัวเลข โดยตัวอย่างที่นิยมใช้คือการทำนายราคาของบ้าน, การทำนายยอดขายสินค้าในอนาคต

Model performance ที่มา

ในส่วนของ supervised learning จะโฟกัสไปที่ปัญหาทาง classification เพื่อจำแนกประเภทของข้อมูล ซึ่งในส่วนของ regression เพื่อทำนายค่าตัวเลขจากโมเดลนั้นจะมีขั้นตอนต่างๆ ที่คล้ายๆกับ classification models ยกเว้นในส่วนของการ import library ซึ่งสามารถศึกษาเพิ่มเติมได้จาก https://scikit-learn.org/stable/supervised_learning.html

Machine learning - classification ที่มา

#Model evaluation

การประเมินแบบจำแนก (classification) ในการเรียนรู้ของเครื่องจักร (machine learning) มีหลายวิธีที่สามารถใช้ได้ โดยแบ่งออกเป็นการประเมินความแม่นยำ (accuracy) และการประเมินผลโมเดลโดยรวม (overall model performance) ซึ่งสามารถทำได้โดยใช้เครื่องมือหลายอย่าง เช่น confusion matrix, precision, recall, F1 score, ROC curve, AUC, และอื่น ๆ

ตัวอย่างเครื่องมือที่ใช้ในการประเมินแบบจำแนกได้แก่:

-Confusion Matrix: ตารางที่แสดงผลลัพธ์จากการจำแนกโดยแบ่งเป็นคลาสต่าง ๆ โดยสามารถคำนวณ precision, recall, F1 score และอื่น ๆ ได้

-Accuracy: ค่าสัมประสิทธิ์ที่แสดงถึงความแม่นยำของโมเดลโดยรวม

-Precision: ค่าสัมประสิทธิ์ที่แสดงถึงความแม่นยำของการทำนายในคลาสบวก (positive class)

-Recall: ค่าสัมประสิทธิ์ที่แสดงถึงความสามารถในการตรวจจับคลาสบวก (positive class)

-F1 Score: ค่าสัมประสิทธิ์ที่เป็นความสมดุลระหว่าง precision และ recall

-ROC Curve (Receiver Operating Characteristic Curve): กราฟที่แสดงความสามารถในการแยกแยะของโมเดลระหว่าง true positive rate (sensitivity) และ false positive rate (1-specificity)

-AUC (Area Under the ROC Curve): ค่าของพื้นที่ใต้เส้นโค้ง ROC ซึ่งสามารถใช้เป็นตัวบ่งชี้ของประสิทธิภาพของโมเดลในการแยกแยะ

$Precision = \frac{TP}{TP+FP}$ \

$Recall = \frac{TP}{TP+FN}$ \

\(F1\text{-}Score = 2 \times \frac{Precision \times Recall}{Precision + Recall}\) \textbf{Accuracy}:

สมการความแม่นยำ (Accuracy): \begin{align} \text{Accuracy} = \frac{\text{TP} + \text{TN}} {\text{TP} + \text{TN} + \text{FP} + \text{FN}} \end{align}

สมการค่า AUC (Area Under the Curve): \begin{align} \text{AUC} = \int_{0}^{1} \text{TPR}(f) , \text{d}\text{FPR}(f) \end{align}

โดยที่:

  • TP: True Positive จำนวนกรณีที่เป็นจริงและทำนายว่าเป็นจริง

  • TN: True Negative จำนวนกรณีที่เป็นเท็จและทำนายว่าเป็นเท็จ

  • FP: False Positive จำนวนกรณีที่เป็นเท็จแต่ทำนายว่าเป็นจริง

  • FN: False Negative จำนวนกรณีที่เป็นจริงแต่ทำนายว่าเป็นเท็จ

  • TPR: True Positive Rate (Sensitivity, Recall)

  • FPR: False Positive Rate (1 - Specificity)

เอกสารอ้างอิงและศึกษาเพิ่มเติม อ้างอิง:

-Bishop, C. M. (2006). Pattern Recognition and Machine Learning. Springer.

-Hastie, T., Tibshirani, R., & Friedman, J. (2009). The Elements of Statistical Learning: Data Mining, Inference, and Prediction. Springer.
